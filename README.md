# About
Life Link is a web-based blood donation management system developed for the Web Development (CSE 242) course at East Delta University. The system enables secure donor registration, authentication-based access, and efficient donor search based on blood group and location.


# Features
- Secure user registration and login using authentication
- Donor profile creation and management
- Search donors by blood group and location
- Authentication-based access to donor contact information
- Donor dashboard to manage availability status
- Relational database design for structured data management
- Role-based access control for data security
- Responsive and user-friendly interface


# Tech Stack
- Frontend: React, TypeScript (TSX), HTML5, CSS3
- Backend: Supabase (Authentication & Database), Role-based access control, Secure API integration
- Deployment: Vercel
